O contador pergunta sobre isenção de ICMS para exportação. Em 3 segundos, o sistema retorna o trecho exato do regulamento — com número do artigo, página e link para o documento original. Sem ler 400 páginas. Sem confiar em memória.

Milhares de documentos fiscais. Uma pergunta em linguagem natural.

Bases fiscais acumulam regulamentos, pareceres, notas técnicas e atualizações constantes. Busca por palavra-chave falha quando a terminologia varia. O desafio: entender a intenção da pergunta e localizar o trecho exato que responde, com rastreabilidade completa.

Busca por significado, não por palavra

Cada documento é fragmentado em trechos semânticos, vetorizado via embeddings e indexado no pgvector. "Isenção de ICMS para exportação" encontra respostas mesmo quando o texto usa "desoneração tributária em operações internacionais".

Citação rastreável até o parágrafo

Toda resposta aponta documento, página e parágrafo de origem. O usuário verifica a fonte com um clique. Isso transforma o sistema de uma caixa-preta em uma ferramenta que o departamento jurídico aceita usar.

Pipeline auditável de ponta a ponta

Retrieval, reranking, generation — cada etapa gera logs estruturados. É possível entender por que uma resposta foi dada e, mais importante, por que outra não foi. Essencial para compliance fiscal.
